# EventSpark

This project is used for the frontend development of EventSpark. Follow is the quick guide for how to use.

## Environment Prepare

Install all dependencies

```bash
yarn
```

### Start project

```bash
yarn start
```

### Build project

```bash
yarn build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
